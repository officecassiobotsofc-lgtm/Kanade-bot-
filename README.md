🚀 Instalação

git clone git@github.com:officecassiobotsofc-lgtm/Kanade-bot-.git
cd Kanade-bot-
npm install

---

▶️ Iniciar

npm start

---

📱 Comandos

Comando| Função
".menu"| Abrir menu
".ping"| Status do bot
".info"| Informações
